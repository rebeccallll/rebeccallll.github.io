AI-powered J.A.R.V.I.S system

J.A.R.V.I.S (Just A Rather Very Intelligent System) is an interactive AI assistant based on the popular character from the Iron Man series. This product is a Unity application that utilizes the OpenAI API to create a dynamic and engaging conversation with users.

Keyword: ChatGPT, API, .Net, C#, unity, NuGet
